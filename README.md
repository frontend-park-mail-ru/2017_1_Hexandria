# 2017_1_Hexandria
Hexandria: strategy game. Technopark front-end project.

<b>Install:</b>
````
git clone https://github.com/frontend-park-mail-ru/2017_1_Hexandria.git
cd 2017_1_Hexandria/
npm install
````

<b>Run Dev:</b>
````
npm sassstartup
npm startup
http://localhost:3000/
````
<b>Production:</b>

https://hexandria.herokuapp.com/
